Project Description

Anyone who has driven to the Babbio Center at Stevens Institute of Technology knows the frustration: circling the garage level by level, not knowing where a spot is available until you have already passed it. The Babbio Parking Garage Simulator was built to solve that problem through a clear, visual representation of real-time parking availability.

The simulation models all five levels of the Babbio garage with 20 spots per level, totaling 100 spots. Each spot can be toggled between available and occupied states, allowing users to simulate vehicle entries and exits. A live alert feed logs recent spot changes, and the home page provides a high-level summary of occupancy across every floor.

This project was completed as part of the Web Programming I course at Stevens Institute of Technology. The entire site is built with semantic HTML5 and external CSS3, with no JavaScript frameworks or backend — demonstrating that a functional, responsive, and accessible web application can be built with the fundamentals alone.
